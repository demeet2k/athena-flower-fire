# AppG — Square Lens / Constructions

- `AppG.S3.a`: `TriaBalancer` — Monitors the three poles' weights and triggers rebalancing when any pole exceeds 50%. Redistributes weight toward centroid.
- `AppG.S3.b`: `PhaseTriadGenerator` — Generates three-phase clock from the i-seed: `{1, e^{i2π/3}, e^{i4π/3}}`. The three roots of unity for the triangle.
- `AppG.S3.c`: `TriadicBinder` — Composes three agents into a collective agent using the ×-seed's binding law. Verifies associativity by computing both orderings.
- `AppG.S3.d`: `DualCertificationGate` — Collects certification from all three poles, requires at least 2 agreeing before allowing action. Records which poles certified and which abstained or dissented.
