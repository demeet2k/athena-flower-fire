# Ch20 — Square Lens / Laws

- `Ch20<0103>.S2.a`: `CollectiveAuthorizationLaw` — A CollectiveAuthor may form only when all participating agents hold compatible AgentContracts (Ch17) and each agent has explicitly consented to shared governance.
- `Ch20<0103>.S2.b`: `MergeOrderLaw` — The MergeGovernor must resolve parallel edits using a deterministic ordering function; non-deterministic merges are rejected as governance violations.
- `Ch20<0103>.S2.c`: `OrganismCoherenceLaw` — An AgentOrganism must maintain a single coherent manuscript state at all times; if agents' internal models diverge beyond a declared tolerance, the organism must pause and reconcile.
- `Ch20<0103>.S2.d`: `VoiceProtectionLaw` — A merge operation that reduces any agent's voice contribution below a declared minimum threshold must be rejected by the MergeGovernor and returned for revision.
