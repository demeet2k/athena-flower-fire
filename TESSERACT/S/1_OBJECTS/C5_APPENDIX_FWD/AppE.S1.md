# AppE — Square Lens / Objects

- `AppE.S1.a`: `QuarterTurnRotor` — The discrete rotation operator `z ↦ iz` generating the 4-cycle `{1, i, −1, −i}`. This is ROSETTA[i].Square.Earth.L0: the bare phase increment as exact quarter-turn. The atomic clock tick of the nervous system.
- `AppE.S1.b`: `PhaseCycleGroup` — The cyclic group `Z₄ = {1, i, −1, −i}` generated by `i`. Four quarter-turns return to identity. This is the discrete orbit structure from which all mixed-radix clocks are built.
- `AppE.S1.c`: `MixedRadixRegister` — A multi-digit counter where each digit position has its own modulus (cycle length). The 21-station orbit uses base-21, pentadic lanes use base-5, quarter-turns use base-4. Digit overflow cascades upward.
- `AppE.S1.d`: `PolygonClosureSeed` — The π-seed's discrete face: `π = lim n·sin(π/n)`. The n-gon approximation to the circle. Each polygon is a discrete clock with n stations; the limit is the continuous circle gear.
