# Ch05 — Square Lens / Objects

- `Ch05<0010>.S1.a` `CanonicalBucket` — The verified mass container holding claims that have passed all admissibility tests and carry no unresolved contradictions, forming the tome's ground truth.
- `Ch05<0010>.S1.b` `FrontierBucket` — The quarantine container holding claims that carry unresolved contradictions or incomplete evidence, isolated from canonical mass to prevent contamination.
- `Ch05<0010>.S1.c` `ParadoxRegime` — The rule set governing how contradictory claims are classified, contained, and either resolved into canonical status or permanently quarantined.
- `Ch05<0010>.S1.d` `ContradictionFilter` — The predicate engine that detects mutual inconsistencies between claims, flagging pairs or sets that cannot simultaneously hold under the current axiom set.
