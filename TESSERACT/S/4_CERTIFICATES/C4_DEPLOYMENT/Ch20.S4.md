# Ch20 — Square Lens / Certificates

- `Ch20<0103>.S4.a`: `CollectiveFormationCert` — A multi-signed certificate recording all participating agents' consent, their compatible AgentContracts, and the shared governance terms of the CollectiveAuthor.
- `Ch20<0103>.S4.b`: `MergeOrderCert` — A proof that a specific merge was executed using the declared deterministic ordering function, with the input edits, ordering decision, and output delta recorded.
- `Ch20<0103>.S4.c`: `OrganismCoherenceCert` — A receipt proving that an AgentOrganism's internal model divergence remained within tolerance for a declared number of synchronization cycles.
- `Ch20<0103>.S4.d`: `VoicePreservationCert` — A certificate proving that every agent's voice contribution in a committed merge exceeds the declared minimum threshold, with per-agent metrics recorded.
