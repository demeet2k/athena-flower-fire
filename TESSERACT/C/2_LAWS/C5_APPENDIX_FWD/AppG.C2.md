# AppG — Cloud Lens / Laws

- `AppG.C2.a`: `ConfidenceTriadLaw` — Triadic confidence is maximized when all three poles have equal weight and equal reliability. Asymmetric confidence reduces consensus quality.
- `AppG.C2.b`: `ConsensusThresholdLaw` — Consensus reliability must exceed declared threshold before action is permitted. Below threshold = abstain.
- `AppG.C2.c`: `PoleRedundancyLaw` — The system must tolerate single-pole failure without total governance collapse. Two remaining poles can operate in degraded mode.
- `AppG.C2.d`: `DegenerationPreventionLaw` — If degeneration risk exceeds threshold, the system must trigger pole restoration or graceful degradation to two-pole mode with declared limitations.
