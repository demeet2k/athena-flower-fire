# Ch21 — Cloud Lens / Laws

- `Ch21<0110>.C2.a`: `ReplicationTruthLaw` — The ReplicationTruthSurface must be monotonically growing; once a seed's viability is certified, that certification may not be retracted without a full re-extraction and re-verification.
- `Ch21<0110>.C2.b`: `SuccessionValidityLaw` — The SuccessionValidityRegion must not expand beyond the bounds set by the parent framework's laws; child frameworks may tighten but never loosen their validity constraints.
- `Ch21<0110>.C2.c`: `ProductCollapseDetectionLaw` — The ProductCollapseDetector must run at every verification epoch; skipping a detection cycle constitutes a system integrity violation.
- `Ch21<0110>.C2.d`: `FrameworkFidelityLaw` — The FrameworkFidelityManifold must be simply connected at FrameworkBirth time; any hole indicates an inherited law that has not been fully implemented in the NextCrystal.
