# Ch05 — Cloud Lens / Laws

- `Ch05<0010>.C2.a` `CanonicalConfidenceMonotonicity` — As new verified claims are added to canonical mass, the aggregate confidence score must increase or hold steady, never decrease.
- `Ch05<0010>.C2.b` `FrontierRiskCeiling` — The frontier risk profile must remain below a declared ceiling; exceeding it triggers emergency quarantine reinforcement.
- `Ch05<0010>.C2.c` `ParadoxUncertaintyIrreducibility` — The paradox uncertainty cloud cannot be reduced below its irreducible core without introducing new axioms or evidence.
- `Ch05<0010>.C2.d` `ContradictionSeverityTruthfulness` — The severity index must faithfully reflect the logical depth of the contradiction; understating severity is a certification violation.
