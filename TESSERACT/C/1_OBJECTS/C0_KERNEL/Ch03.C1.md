# Ch03 — Cloud Lens / Objects

- `Ch03<0002>.C1.a` `CorridorTruthGradient` — The graded measure of truth certainty along a corridor's length, highest at source and potentially degrading toward the destination.
- `Ch03<0002>.C1.b` `WitnessReliabilityScore` — The quantified trustworthiness of a witness based on historical attestation accuracy, used to weight gate-pass verdicts.
- `Ch03<0002>.C1.c` `CompressionUncertainty` — The residual information gap between a compressed representation and its source, bounded by the compression bond's fidelity guarantee.
- `Ch03<0002>.C1.d` `AdmissibilityConfidenceInterval` — The statistical range within which the true admissibility verdict lies, given the available evidence and its measurement precision.
