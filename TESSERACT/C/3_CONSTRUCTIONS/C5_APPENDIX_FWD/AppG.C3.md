# AppG — Cloud Lens / Constructions

- `AppG.C3.a`: `TriadicConfidenceEstimator` — Estimates the balance confidence field from pole weight and reliability measurements.
- `AppG.C3.b`: `ConsensusReliabilityComputer` — Computes consensus reliability from individual pole reliabilities and the dual-certification gate logic.
- `AppG.C3.c`: `PoleFailureMonitor` — Monitors each pole for signs of failure, drift, or unreliability. Triggers alerts when failure probability exceeds threshold.
- `AppG.C3.d`: `DegenerationPreventor` — Triggers pole restoration or graceful degradation when degeneration risk exceeds threshold.
