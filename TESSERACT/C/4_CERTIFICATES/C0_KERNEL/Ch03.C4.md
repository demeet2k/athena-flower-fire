# Ch03 — Cloud Lens / Certificates

- `Ch03<0002>.C4.a` `Cert_Gradient_Non_Inverted` — Attests that the truth gradient along the corridor was measured and found monotonically non-increasing from source to destination.
- `Ch03<0002>.C4.b` `Cert_Reliability_Scored` — Attests that a witness reliability score was computed from verified historical data with the decay model and parameters disclosed.
- `Ch03<0002>.C4.c` `Cert_Uncertainty_Bounded` — Attests that compression uncertainty was measured and falls below the declared threshold, with the gap analysis attached.
- `Ch03<0002>.C4.d` `Cert_Confidence_Narrowed` — Attests that the admissibility confidence interval narrowed upon incorporation of new evidence, with before and after measurements.
