# Ch03 — Fractal Lens / Objects

- `Ch03<0002>.R1.a` `CorridorRecursionTree` — The recursive decomposition of a truth corridor into sub-corridors, where each sub-corridor enforces the same admissibility laws as the whole.
- `Ch03<0002>.R1.b` `WitnessNestingChain` — The chain of nested witness obligations where a gate's witness may itself require witnesses, forming a recursive attestation tree.
- `Ch03<0002>.R1.c` `CompressionDepthStack` — The stack of successive compression layers applied to a claim, where each layer's bond references the layer above it.
- `Ch03<0002>.R1.d` `AdmissibilitySelfReference` — The property that the admissibility envelope's own definition must pass its own admissibility test, ensuring the predicate is self-consistent.
