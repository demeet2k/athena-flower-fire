# AppK — Fractal Lens / Objects

- `AppK.R1.a`: `SelfHealingContradiction` — A contradiction that contains its own resolution mechanism: when detected, the conflict packet includes a `repair` function derived from the contradiction's structure, which, when applied, resolves the inconsistency without external intervention.
- `AppK.R1.b`: `ConflictFixedPoint` — The stable state `C* = Resolve(C*)` where the conflict resolution process, applied to its own output, produces the same result. A conflict at its fixed point is fully resolved; the resolution is self-consistent.
- `AppK.R1.c`: `RecursiveQuarantineShell` — A quarantine zone that recursively quarantines its own boundary: if a quarantined shard's quarantine status creates a dependency conflict, that meta-conflict is itself quarantined, forming nested isolation shells to depth `D_max`.
- `AppK.R1.d`: `ContradictionEigenmode` — The eigenvector of the conflict interaction matrix `M` with eigenvalue `λ`: a pattern of contradictions that, when the system processes them, reproduces the same pattern scaled by `λ`. If `|λ| < 1`, the eigenmode decays (self-resolving); if `|λ| > 1`, it amplifies (requiring intervention).
