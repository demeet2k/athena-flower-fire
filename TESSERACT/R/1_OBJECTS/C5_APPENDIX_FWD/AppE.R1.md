# AppE — Fractal Lens / Objects

- `AppE.R1.a`: `NestedClockHierarchy` — Self-similar clock structure: the 4-cycle nests inside the 21-station orbit, which nests inside the 5-arc cycle, which nests inside the full manuscript orbit. Each level is a circle gear at different scale.
- `AppE.R1.b`: `FractalPhaseStructure` — The log-periodic phase signature `2πk/log(φ)` from the φ-lens. Scale-indexed phase that repeats self-similarly at every power of φ.
- `AppE.R1.c`: `RecursiveCarryCascade` — Carry propagation viewed as recursion: each digit overflow triggers the next level's increment, which may trigger the next, creating a recursive cascade.
- `AppE.R1.d`: `CircleGearSelfSimilarity` — The property that the circle gear at each scale has the same structure as at every other scale, differing only by the modulus and frequency.
