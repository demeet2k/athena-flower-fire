# Ch20 — Fractal Lens / Objects

- `Ch20<0103>.R1.a`: `RecursiveAuthoring` — A collective authoring structure in which each agent in the CollectiveAuthor is itself a CollectiveAuthor of sub-agents, enabling hierarchical manuscript production.
- `Ch20<0103>.R1.b`: `FractalMerge` — A merge operation where the content being merged itself contains nested merge results from lower-level CollectiveAuthors, requiring recursive conflict resolution.
- `Ch20<0103>.R1.c`: `SelfGoverningCollective` — A CollectiveAuthor that contains its own MergeGovernor as a participating agent, enabling the governance itself to be collectively authored and modified.
- `Ch20<0103>.R1.d`: `DepthIndexedVoice` — A voice contribution annotated with the recursion depth of the CollectiveAuthor that produced it, enabling voice preservation analysis at every nesting level.
