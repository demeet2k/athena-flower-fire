# AppG — Fractal Lens / Constructions

- `AppG.R3.a`: `RecursiveTriadicBuilder` — Constructs nested tria prima to declared recursion depth. Each pole receives its own internal governance.
- `AppG.R3.b`: `SierpinskiDecomposer` — Decomposes governance triangle into sub-triangles, each with independent governance. Tracks center-removal and sub-triangle relationships.
- `AppG.R3.c`: `RecursiveConsensusAggregator` — Aggregates consensus certificates from sub-levels into parent-level consensus. Propagates upward.
- `AppG.R3.d`: `TriadicScaleBridgeBuilder` — Constructs the ScaleBridge between two governance levels using φ-lens frequency rescaling.
