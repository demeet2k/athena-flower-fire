# Ch03 — Fractal Lens / Laws

- `Ch03<0002>.R2.a` `CorridorRecursionInvariance` — Sub-corridors must satisfy the same admissibility axioms as the parent corridor at every recursion depth.
- `Ch03<0002>.R2.b` `WitnessNestingTermination` — The witness nesting chain must terminate at a finite depth where base witnesses require no further attestation.
- `Ch03<0002>.R2.c` `CompressionDepthBound` — The compression depth stack has a maximum depth; exceeding it requires explicit authorization and a fidelity re-guarantee.
- `Ch03<0002>.R2.d` `AdmissibilitySelfConsistency` — The admissibility predicate must pass its own test; a predicate that fails its own criteria is automatically disqualified.
