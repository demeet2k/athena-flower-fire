# AppK — Fractal Lens / Laws

- `AppK.R2.a`: `SelfHealingConvergenceLaw` — Self-healing contradictions must converge: the embedded repair function is a contraction with `κ < 1`, guaranteeing resolution within `⌈log(δ_0/ε)/log(1/κ)⌉` iterations by Banach's theorem, mirroring AppJ's contraction law.
- `AppK.R2.b`: `FixedPointConsistencyLaw` — The conflict fixed point must be unique and consistent with all six conservation laws (AppB). A resolution that satisfies itself but violates conservation is not a true fixed point; it is a pseudo-resolution that will re-trigger detection.
- `AppK.R2.c`: `RecursiveQuarantineDepthBound` — Recursive quarantine nesting is bounded at depth `D_max = 6` (one per conservation law). Deeper nesting indicates a fundamental structural problem requiring organism-level intervention rather than local containment.
- `AppK.R2.d`: `EigenmodeStabilityLaw` — All contradiction eigenmodes must have `|λ| ≤ 1`: no amplifying eigenmode is permitted in the steady state. If `|λ| > 1` is detected, the corresponding contradiction pattern must be suppressed by targeted quarantine of the eigenmode's support.
