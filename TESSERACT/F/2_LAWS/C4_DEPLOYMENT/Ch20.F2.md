# Ch20 — Flower Lens / Laws

- `Ch20<0103>.F2.b`: `MergeDynamicLaw` — The MergeDynamic must complete its full four-phase cycle for each batch of parallel edits; skipping phases produces incomplete provenance.
- `Ch20<0103>.F2.c`: `OrganismGrowthLaw` — An AgentOrganism must not divide into sub-organisms while any merge conflict remains unresolved; division requires a clean coherence state.
